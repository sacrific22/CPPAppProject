<h3 align="left">Main Idea of the project:</h3>

###

<h5 align="left">Develop a catch'a'fruit game using c++ with score system and random points gain.</h5>

###

<h3 align="left">How to install and run:</h3>

###

<h5 align="left">Download this files:<br>https://github.com/libsdl-org/SDL/releases/tag/release-2.32.8<br>https://github.com/libsdl-org/SDL_ttf/releases/tag/release-2.24.0<br>https://github.com/libsdl-org/SDL_image/releases/tag/release-2.8.3<br>https://www.ambiera.com/irrklang/<br>After download you will need to set them in Visual studio(better drop them in a lib folder in project)<br>Path to set them<br>Project -> Project settings -> Choose configure ALL -> C++ -> Add additional catalogues(just put the include folder as a path of each library) then go -> Language -> C++ 17 -> then go Compositor (Linker) -> Additional catalogues of libraries -> and set <name of library><version>/lib/x64 folder as path. <br>Stay in Compositor -> Input -> and put .lib files as a path<br>After that program will be ready to start.</h5>

###

<h3 align="left">Controls:</h3>

###

<p align="left">Game:<br>Shift - Increase speed of player<br>Left arrow - move left<br>Right arrow - move right<br>Menu:<br>Up arrow - switch up in menu<br>Arrow down - switch down in menu<br>Settings menu<br>S - Decrease volume in settings<br>W - Increase volume in settings<br>A - Choose previous skin<br>D - Choose next skin <br>Left Arrow - Decrease brightness<br>Right Arrow - Increase brightness</p>

###

<h3 align="left">Goal in game:</h3>

###

<h5 align="left">Reach 500 points to win!</h5>

###

<h3 align="left">Program architecture:</h3>

###

<p align="left">/assets/ - contains fonts, sprites, skins, background images, music.<br>/lib/ - contains libraries<br>/src/ - contains code files<br>main.cpp - main logic and screen switching<br>game.cpp - game logic<br>game.h - game logic</p>
